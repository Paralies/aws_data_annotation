##  주요 기능
### 결과 관리
- summary
- search 
- all annotator
### DB 관리
1. `DB/meta.json` update
2. `Refresh DB` 버튼 클릭 -> platform에 반영됨
### Annotator 관리
- Register Annotator
- ID, password 를 관리자가 부여해서 전달 
    - password는 사용자가 변경 가능
- age, gender도 관리자가 등록 시 입력
- 등록된 사용자들은 `logs/annotators.json` 에 저장됨
